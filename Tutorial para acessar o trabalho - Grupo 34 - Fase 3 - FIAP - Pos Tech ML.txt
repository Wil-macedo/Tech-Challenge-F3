Instruções

1 - O arquivo "Modelo de ML de detecção de fraudes.ipynb" no GitHub contém todo o notebook de desenvolvimento do modelo de Machine Learning.


2 - Execute o comando para instalar as dependências:
    pip install -r requirements.txt


3 - Para visualizar a aplicação Streamlit alimentada pelo modelo, abra o arquivo "dashboard.py" disponível no GitHub em alguma IDE e 
execute o seguinte comando em um novo terminal:
    "python -m streamlit run dasboard.py", para ser direcionado até o site da aplicação em uma localhost


4 - O deploy do dashboard foi realizado em:
    tcfase3-embnfnazcnf4gqhf.brazilsouth-01.azurewebsites.net
    É uma App Service gratuita, então pode apresentar instabilidades.


5 - Os conectores Python para SQL e a autenticação com o banco de dados estão disponíveis no código do repositório do GitHub.
